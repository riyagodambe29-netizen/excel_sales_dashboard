My 1st 📊 #Excel Sales Dashboard

An interactive **Sales Dashboard** built in **Microsoft Excel** to analyze sales performance using a sample dataset of **500 sales employees**. The dashboard provides meaningful business insights through dynamic charts, Pivot Tables, Pivot Charts, and Slicers.

## 🚀 Features

- 📈 Total Sales KPI
- 🎯 Target Achievement Analysis
- 🌍 Region-wise Sales Performance
- 👥 Top 5 Sales Executives
- 📉 Bottom 5 Sales Executives
- 🎛️ Interactive Slicers
- 📊 Pivot Tables & Pivot Charts
- 📋 Dynamic Dashboard Layout

---

## 📂 Dataset

This project uses a **sample sales dataset of 500 employees** for learning and portfolio purposes.

The dataset includes:

- Employee Code
- Sales Executive
- Manager
- Region
- City
- Daily Sales (Day 1–Day 31)
- Total Sales
- Sales Target
- Target Achievement (%)

## 📥 How to Use

1. Clone or download this repository.
2. Open **Excel_Sales_Dashboard.xlsx** in Microsoft Excel.
3. If using the `.xlsm` version, enable macros when prompted.
4. Use the slicers to interact with the dashboard and explore sales performance.
